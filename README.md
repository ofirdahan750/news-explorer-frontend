### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ofirdahan750/news-explorer-frontend.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run on a local Terminal
   ```sh
   npm start
   ```
